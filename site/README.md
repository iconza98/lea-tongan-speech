# site/

Public website (Firebase Hosting). Three v1 surfaces:

- **Contribute your voice** — record → show consent ([`../docs/CONSENT.md`](../docs/CONSENT.md))
  → grant CC BY 4.0 → upload to the open-corpus bucket + a `pending` Firestore doc.
- **Dataset browser / download** — browse and download the published CC BY 4.0 corpus.
- **Eval leaderboard** — model scorecards over time (automated metrics now, MOS later).

No live GPU synth demo in v1 (cost/abuse). *TBD — Phase 1–2.*
